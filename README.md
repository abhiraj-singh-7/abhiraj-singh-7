## Featured Project

### calMOG — AI Syllabus to Calendar App

**Live:** https://www.calmog.net  
**Repo:** Private

calMOG helps students upload course syllabi and automatically extract assignments, quizzes, exams, grading weights, and course policies into a structured academic calendar.

Built with:

- Next.js
- TypeScript
- Supabase Auth
- Supabase Postgres + RLS
- Supabase Storage
- OpenAI API
- PDF text extraction
- Vercel

Core features:

- Email/password authentication
- Per-user course and deadline data
- AI syllabus extraction
- Human-in-the-loop review before adding deadlines
- Calendar dashboard
- Pending review workflow
- Custom domain deployment
